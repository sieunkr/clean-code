````java
public List<RecoredGrip> retrievSection(String sectionName) {

        stream.close();

        return new StorageException("..");
    }
}
````

외부 API 를 감싸면 외부 라이브러리와 프로그램 사이에서 의존성이 크게 줄어든다. 나중에 다른 라이브러리로 갈아타도 비용이 적다.
